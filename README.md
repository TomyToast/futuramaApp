# Futurama APP

## Main Technologies:
- React,
- Sass

### Functionalities:
- Using data from Futurama API ("https://futuramaapi.herokuapp.com/")
- Displays ten picked heroes with name, image and one random            quote
- Displays up to five quotes after clicked on character
- App is responsive

### Start-up in root folder:
- " npm i "
- " npm start "
